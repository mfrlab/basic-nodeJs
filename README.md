# basic-nodeJs
Basic code for support big project that can easyly to use.
Tech stack using Javasript, Sql, Sequelize, ExpressJS

# pre-Requisite
```
- Node JS
- NPM
- MySQL
```

# Getting Started
- Clone Project
```
move to your local directory
Git Clone <project repository on git>
```
- Install Dependencies
```
npm install
```
- Setup Environment
```
setup file .env
```
- Running Project
```
npm run <script>
```
